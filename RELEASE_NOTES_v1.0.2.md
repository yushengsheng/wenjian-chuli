# 表格文件处理工具 v1.0.2

本次发布聚焦于处理链路稳定性、预览一致性，以及旧版本辅助列兼容。

本次内容：
- 预览区默认支持全量显示，并新增“仅预览变动”切换
- 修改前 / 修改后行数改为显示实际总行数与当前显示行数，避免未写回时误判无变化
- 字段匹配改为严格按首行标题同名匹配；首行标题为空的列直接忽略
- `仅更新不新增` / `仅用新数据补全老数据空值` 下，不再保留空主键新行
- 对比预览不再把来源文件、来源工作表、数据分组、导入顺序这类辅助列当作业务变动
- 兼容旧版本已经写入老表的中文辅助列，并在后续处理时将其识别为系统字段
- 修复主窗口被直接销毁时后台线程未清理、进程可能挂住的问题
- 使用全新虚拟环境复跑测试与打包，补充 `jinja2` 构建依赖，确保便携包可直接在其他 Windows 设备使用
