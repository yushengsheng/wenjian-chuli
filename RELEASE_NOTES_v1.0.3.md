# 表格文件处理工具 v1.0.3

本次发布重点修复主键合并、对比预览和写回保留结构的问题，并优化大数据量场景下的处理体验。

本次内容：
- 修复 `更新并新增` / `仅更新不新增` 下新数据空值覆盖老数据已有值的问题
- 修复空主键新行被错误用于更新或追加的问题，仅在主键满足条件时才更新
- 调整多选主键匹配逻辑：多个主键字段时，任意一个命中即可匹配更新；新值为空仍不会覆盖老值
- 修复新数据内部重复主键在无老记录匹配时被错误压成一行、导致字段静默丢失的问题
- 修复修改对比预览在主键列隐藏或大量空主键场景下的错位显示，避免把少量变更误显示为整片新增/移除
- 原始预览与修改对比预览均新增行号列，便于人工核对
- 兼容部分样式损坏的 Excel 文件导入，自动修复非法空填充样式后继续读取
- 写回老数据 / 导出完整老文件时，改为尽量保留目标工作表结构与格式，不再整张删除重建
- 优化对比预览性能，数千行数据下的处理与比对明显加快
- 新增底部处理中进度条，处理、导出、写回时可直观看到程序仍在运行
